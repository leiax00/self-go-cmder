# self-go-cmder
> 用于日常小工具整理, 同时穿插部分个人练习

# 目录说明
```text
self-go-cmder
      |---- cmds                          # 命令操作实现集
      |     |---- practice                # 个人练习操作集
      |     |     |---- index.go          # 操作集组装, 面向conf
      |     |     |---- ...               # 操作具体实现
      |     |---- usual                   # 常用操作集
      |     |     |---- index.go          # 操作集组装, 面向conf
      |     |     |---- ...               # 操作具体实现        
      |     |---- root.go
      |---- conf                          # 命令配置汇总
      |---- domain                        # 命令对象定义
      |---- go.mod
      |---- main.go
```

# 功能集说明
1. 执行命令: `go run main.go [cmd]`

2. 可编译成可执行程序使用: 
    ```shell
    go build
    self-go-cmder [cmd]
    ```

|  序号 |         命令                   |                 功能                                        |              备注           |
